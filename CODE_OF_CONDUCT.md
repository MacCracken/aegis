# Code of Conduct

## Our Standards

- Use welcoming and inclusive language.
- Be respectful of differing viewpoints.
- Accept constructive criticism gracefully.
- Focus on what is best for the project.

## Scope

This applies to all project spaces: GitHub issues, pull requests, code reviews, and any other channel where the project is discussed.

## Enforcement

Report incidents to **cyriusmaccken@gmail.com**. Reports will be reviewed and acted on promptly. Maintainers may warn, temporarily ban, or permanently ban participants who violate these standards.
